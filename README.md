 ## 環境建置：
 1. Mongodb
 2. Node.js
 
 ## 下載方法：
 1. 打開終端機，Clone 此專案至本機電腦
 
```https://github.com/haru5386/password_check```

2. 進入存放此專案的資料夾

```cd password_check```

3. 安裝 npm 套件，

```npm install```

4. 加入種子資料

```npm run seed```

5. 啟動網頁伺服器

```npm run dev```

6. 顯示`Express is running on http://localhost:3000，mongodb connected`
   表示成功進入
